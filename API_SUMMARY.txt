CRITICAL GAPS:

🔴 SEVERE (Likely blocking for some use cases)
  1. TypeScript: No snapshot status check (exists in FFI, not exposed)
     → photostax_snapshot_check_status not called from TS layer
  
  2. ImageRef: No path/file access in any binding
     → Cannot determine source file location
  
  3. ImageRef: No deletion support
     → Cannot delete image files

🟡 MODERATE (Nice-to-have features)
  4. No QueryResult cursor-based iteration
     → Must paginate instead
  
  5. No PhotoStack.repo_id in bindings
     → Cannot identify repository source
  
  6. No image streaming (only full read)
     → Cannot process large images efficiently
  
  7. No event subscriptions (Core has them)
     → Cannot react to cache changes
